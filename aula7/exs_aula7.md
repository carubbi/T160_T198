# Exercícios Beecrowd: Estruturas de Repetição

## Objetivo
Esta lista reúne exercícios da categoria **Iniciante** do Beecrowd que podem ser trabalhados com estruturas de repetição, considerando o que os alunos já estudaram até agora:

- variáveis e operadores
- entrada, processamento e saída
- `if`
- `if...else`
- `if...else if...else`
- `switch-case`
- condicionais aninhadas
- laços `while`, `do...while` e `for`

O foco é praticar repetição sem exigir, neste momento:

- vetores
- matrizes
- funções
- ordenação
- recursão
- estruturas de dados

## 1. Exercícios indicados para `while`
São mais adequados quando a repetição depende de uma condição de parada que vai sendo descoberta durante a execução.

- `1114` Senha Fixa
- `1115` Quadrante
- `1116` Dividindo X por Y
- `1117` Validação de Nota
- `1134` Tipo de Combustível
- `1154` Idades

## 2. Exercícios indicados para `do...while`
O Beecrowd normalmente não exige explicitamente `do...while`, mas alguns problemas podem ser resolvidos com essa estrutura quando faz sentido executar o bloco pelo menos uma vez antes de validar a continuação.

- `1114` Senha Fixa
- `1117` Validação de Nota
- `1134` Tipo de Combustível
- `1154` Idades

## 7. Observações didáticas
- `1114` e `1117` são ideais para introduzir `while` e `do...while`.
- `1078`, `1073` e `1072` são ideais para introduzir `for`.
- `1095` a `1098` são bons para treino de rastreamento de variáveis e teste de mesa.
- `1094` e `1154` são bons para trabalhar acumuladores e médias.
- `1151` já exige mais atenção com atualização de variáveis em sequência.
