# Exercícios Beecrowd: Aplicação Correta de Estruturas Condicionais

## Objetivo
Esta lista reúne exercícios da categoria **Iniciante** do Beecrowd adequados para a Aula 5, cujo foco é escolher corretamente entre:

- `if`
- `if...else`
- `if...else if...else`
- `switch-case`

O objetivo não é apenas resolver o problema, mas decidir qual estrutura é mais adequada para cada tipo de decisão.

## 8. Observações didáticas
- `1046` é bom para discutir quando `if...else` basta.
- `1037` e `1051` são bons para mostrar por que a ordem dos testes importa.
- `1050`, `1052` e `1038` são ideais para justificar o uso de `switch-case`.
- `1041` e `1043` ajudam a comparar soluções com `if...else if...else` e `switch-case`, quando aplicável.
